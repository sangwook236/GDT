[-] Installation (Windows).
	- Install.
		Download GetGnuWin32:
			http://getgnuwin32.sourceforge.net

		Run GetGnuWin32-0.6.3.exe

		Run command prompt.

		cd ${GetGnuWin32_HOME}
		download.bat
		install.bat ${GNUWIN32_INSTALL_DIR}
			install.bat ../gnuwin32
